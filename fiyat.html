Nasıl Kullanılır? Tarayıcılar güvenlik (CORS) nedeniyle dış sitelerden doğrudan arka planda veri kazımasına izin vermez. Gerçek zamanlı ve hatasız bir web kazıma altyapısı için tek yapmanız gereken aşağıdaki adımları izlemektir:
  1. Bilgisayarınızda terminale girip şu komutu çalıştırın: pip install flask requests beautifulsoup4
  2. Aşağıdaki Python kodunu kopyalayıp bir app.py dosyası olarak kaydedin ve python app.py ile çalıştırın.
  3. Tarayıcınızdan http://localhost:5000 adresine girerek bu arayüzü tam fonksiyonel olarak kullanın.

Python Sunucu Kodu (app.py)

Aşağıdaki kodu kopyalayıp bilgisayarınızda app.py adıyla kaydedin:

from flask import Flask, jsonify, render_template_string, request
import requests
from bs4 import BeautifulSoup

app = Flask(__name__)

# Tek dosya içinde hem sunucu hem arayüz sunumu
HTML_TEMPLATE = """
<!-- Bu arayüz yukarıdaki HTML/JS kodunun birebir aynısıdır -->
"""

def fetch_market_data(keyword):
    try:
        # marketfiyati.org.tr veya arama motoru üzerinden web scraping
        url = f"https://www.google.com/search?q=site:marketfiyati.org.tr+{keyword}"
        headers = {'User-Agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64)'}
        response = requests.get(url, headers=headers, timeout=5)
        
        if response.status_code == 200:
            soup = BeautifulSoup(response.text, 'html.parser')
            result_div = soup.find('div', class_='BNeawe')
            if result_div:
                return result_div.text
        return "Eşleşen veri bulunamadı."
    except Exception as e:
        return f"Bağlantı hatası: {str(e)}"

@app.route('/api/search')
def api_search():
    keyword = request.args.get('q', '')
    data = fetch_market_data(keyword)
    return jsonify({'result': data})

if __name__ == '__main__':
    app.run(port=5000, debug=True)

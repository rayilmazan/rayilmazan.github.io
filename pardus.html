Pardus e-Tahta Komut Paneli

Akıllı tahta arıza ve yönetim işlemleri için hızlı kopyalanabilir komutlar

Ağ Servisini Yeniden Başlatma
İnternet veya IP alma sorunlarında NetworkManager'ı tetikler.
sudo systemctl restart NetworkManager
Paket Listesini Güncelleme
Depoları tazeler ve sistem paket listesini eşitler.
sudo apt update
Sistem Yükseltme (Upgrade)
Mevcut paketleri ve uygulamaları günceller.
sudo apt upgrade -y
Boş Alan / Disk Temizliği
Gereksiz önbellek kalıntılarını ve artık paketleri temizler.
sudo apt clean && sudo apt autoremove -y
Dokunmatik Servisini Yeniden Başlat
Dokunmatik algılamayan tahtalarda sürücüyü tetikler.
sudo systemctl restart etrust-touch
Grafik Arayüzü Yeniden Başlatma
Donma durumunda masaüstü oturumunu baştan yükler.
sudo systemctl restart lightdm
